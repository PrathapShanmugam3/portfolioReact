# **App Name**: Prathap's Portfolio

## Core Features:

- Project Showcase: Display detailed information about each project, including tech stack, code snippets, and live demos, sourced directly from provided links such as GitHub, GitLab and Vercel
- Skill Visualization: Visually represent technical skills and proficiency levels, updating automatically from project data.
- Interactive Resume: Enable users to dynamically interact with the resume, filtering experiences and skills based on criteria. Generate various exportable formats.
- Design Theme Customization: Allow users to choose from pre-designed themes or customize colors and layouts to match their personal preferences. Themes can affect the background and other aesthetic styles.
- AI-Powered Personalization: Implement an AI tool that tailors the portfolio content to match specific job descriptions, highlighting relevant skills and experiences.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey professionalism and innovation.
- Background color: Light Gray (#F0F0F0), providing a clean and modern backdrop that ensures readability and focuses attention on the content.
- Accent color: Electric Purple (#BF00FF), used to highlight interactive elements and call-to-action buttons, ensuring they stand out.
- Body text font: 'Inter', a grotesque sans-serif, for a neutral and machined modern feel.
- Headline font: 'Space Grotesk', a proportional sans-serif, giving a techy, scientific feel
- Use a consistent set of minimalist icons for skills, technologies, and project categories.
- Employ a responsive, grid-based layout to ensure the portfolio is accessible and visually appealing on all devices.
- Incorporate subtle, tasteful animations to enhance user engagement, such as transitions and hover effects, while maintaining a professional tone.
